서버 정보
  • - CSHians는 웹티즌 리눅스호스팅으로 동작하고 있습니다.
  • - 호스팅 요금은 걱정하실 필요가 없습니다.
  • - 업데이트를 하기 위해서는 GitHub에 Pull Request를 주셔도 되고, FTP로 직접 수정이 가능합니다. GitHub에 요청을 남기신 경우 요청을 Commit함과 동시에 FTP에 반영해 드립니다. FTP로 직접 수정하신 경우 정기 동기화 작업에 의해 GitHub에 반영되오니 굳이 GitHub의 소스도 수정하실 필요는 없습니다.
  • - 사이트 소스의 GitHub 동기화 요청 및 FTP 작업 요청 또한 개발 요청을 하시면 됩니다.
  • - FTP 주소: ftp.cshians.net / ID: cshians / 비밀번호는 오프라인으로 제공합니다.
  • - 일반적인 업데이트에 있어서 필요하지 않겠지만, DB에 접속하시려면 CPanel에서 PHPMyAdmin을 선택하시면 됩니다. ID와 비밀번호는 FTP와 동일합니다.
